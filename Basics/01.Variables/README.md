# Python 变量教程 

## 1. 什么是变量？

变量就像一个盒子，用来存储数据。我们可以给这个盒子起个名字，方便以后使用。

### 生活中的类比
想象你在整理房间：
- 你有一个盒子，用来装书，你在盒子上贴个标签"书"
- 你有一个盒子，用来装衣服，你在盒子上贴个标签"衣服"
- 变量就是这个"盒子"，变量名就是这个"标签"

### 创建变量
```python
name = "小明"      # 创建一个叫name的变量，存储"小明"
age = 18          # 创建一个叫age的变量，存储18
height = 1.75     # 创建一个叫height的变量，存储1.75
is_student = True # 创建一个叫is_student的变量，存储True
```

## 2. 变量命名规则

### 必须遵守的规则
1. **只能使用字母、数字、下划线**
   - 正确：`user_name`, `age2`, `_private`
   - 错误：`user-name`, `@name`, `name!`

2. **不能以数字开头**
   - 正确：`age2`, `name1`
   - 错误：`2name`, `1age`

3. **区分大小写**
   - `name` 和 `Name` 是两个不同的变量
   - `age` 和 `AGE` 是两个不同的变量

4. **不能使用Python关键字**
   - 错误：`if`, `for`, `while`, `class`, `def` 等

### 好的命名习惯
1. **使用有意义的名字**
   ```python
   # 好的命名
   student_name = "张三"
   student_age = 18

   # 不好的命名
   x = "张三"
   y = 18
   ```

2. **使用下划线分隔多个单词**
   ```python
   # 推荐（蛇形命名法）
   user_name = "张三"
   user_age = 18

   # 也可以，但不推荐
   userName = "张三"
   userAge = 18
   ```

3. **避免使用中文**
   ```python
   # 不推荐
   姓名 = "张三"

   # 推荐
   name = "张三"
   ```

## 3. 变量赋值

### 单个变量赋值
```python
x = 10
print("x =", x)  # 输出: x = 10
```

### 多个变量赋相同的值
```python
a = b = c = 0
print("a =", a, "b =", b, "c =", c)  # 输出: a = 0 b = 0 c = 0
```

### 多个变量赋不同的值
```python
x, y, z = 1, 2, 3
print("x =", x, "y =", y, "z =", z)  # 输出: x = 1 y = 2 z = 3
```

### 交换两个变量的值
```python
a = 10
b = 20
print("交换前: a =", a, "b =", b)
a, b = b, a  # 交换
print("交换后: a =", a, "b =", b)
```

### 变量可以重新赋值
```python
name = "小明"
print("第一次:", name)  # 输出: 小明
name = "小红"  # 重新赋值
print("第二次:", name)  # 输出: 小红
name = 100     # 甚至可以改变类型
print("第三次:", name)  # 输出: 100
```

## 4. 变量的数据类型

变量可以存储不同类型的数据：

### 字符串（str）
用引号包围的文字
```python
name = "小明"
message = '你好，世界'
```

### 整数（int）
没有小数的数字
```python
age = 18
count = 100
```

### 浮点数（float）
有小数的数字
```python
height = 1.75
price = 3.14
```

### 布尔值（bool）
True 或 False
```python
is_student = True
is_adult = False
```

## 5. 实践练习

### 练习1：创建个人信息
```python
# 创建变量存储个人信息
name = "张三"
age = 20
height = 1.75
is_student = True

# 打印个人信息
print("姓名:", name)
print("年龄:", age)
print("身高:", height)
print("是学生:", is_student)
```

### 练习2：计算器
```python
# 创建两个数字变量
num1 = 10
num2 = 5

# 计算
print("加法:", num1 + num2)   # 15
print("减法:", num1 - num2)   # 5
print("乘法:", num1 * num2)   # 50
print("除法:", num1 / num2)   # 2.0
```

### 练习3：交换变量
```python
# 交换两个变量的值
a = "苹果"
b = "香蕉"

print("交换前:", a, b)
a, b = b, a
print("交换后:", a, b)
```

## 6. 常见问题解答

### Q1: 为什么变量名不能以数字开头？
**A:** 因为Python需要区分变量名和数字。如果允许数字开头，`2name` 可能是变量名，也可能是数字 2 乘以 name。

### Q2: 变量名用中文可以吗？
**A:** 技术上可以，但不推荐。因为：
- 在不同环境下可能显示有问题
- 输入中文比较麻烦
- 不符合编程习惯

### Q3: 变量可以改变类型吗？
**A:** 可以！Python是动态类型语言，变量可以随时改变类型。
```python
x = 10      # x是整数
x = "hello" # x变成字符串
x = 3.14    # x变成浮点数
```

### Q4: 什么是好的变量名？
**A:** 好的变量名应该：
- 有意义，能看出变量的用途
- 使用英文（推荐）
- 使用下划线分隔多个单词
- 不要太长，也不要太短
