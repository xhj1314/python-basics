# Python 数字教程 

## 1. 什么是数字？

数字是Python中最基本的数据类型之一，用于表示各种数值。

### Python有三种数字类型：
1. **整数（int）**：没有小数的数字，如 10, -3, 0
2. **浮点数（float）**：有小数的数字，如 3.14, -0.5
3. **复数（complex）**：由实部和虚部组成，如 3+4j

## 2. 整数（int）

### 什么是整数？
整数是没有小数部分的数字，可以是正数、负数或零。

### 创建整数
```python
a = 10      # 正整数
b = -3      # 负整数
c = 0       # 零

print("整数示例:", a, b, c)
```

### 整数运算
```python
print("加法:", 10 + 3)   # 13
print("减法:", 10 - 3)   # 7
print("乘法:", 10 * 3)   # 30
print("整除:", 10 // 3)  # 3
print("取模:", 10 % 3)   # 1
```

### 整数的特点
- 可以是任意大小，没有限制
- 可以是正数、负数或零
- 支持各种算术运算

## 3. 浮点数（float）

### 什么是浮点数？
浮点数是带有小数部分的数字。

### 创建浮点数
```python
pi = 3.14          # 正浮点数
temperature = -0.5  # 负浮点数
scientific = 2.5e3  # 科学计数法（2500.0）

print("浮点数示例:", pi, temperature, scientific)
```

### 浮点数运算
```python
print("加法:", 3.14 + 1)      # 4.14
print("减法:", 3.14 - 1)      # 2.14
print("乘法:", 3.14 * 2)      # 6.28
print("除法:", 3.14 / 2)      # 1.57
```

### 注意：浮点数精度问题
```python
result = 0.1 + 0.2
print("0.1 + 0.2 =", result)  # 可能不是精确的0.3
```

## 4. 复数（complex）

### 什么是复数？
复数由实部和虚部组成，虚部以字母 j 结尾。

### 创建复数
```python
z = 3 + 4j  # 实部是3，虚部是4

print("复数:", z)
print("实部:", z.real)  # 3.0
print("虚部:", z.imag)  # 4.0
```

### 复数运算
```python
z1 = 1 + 2j
z2 = 3 + 4j
print("加法:", z1 + z2)  # (4+6j)
print("乘法:", z1 * z2)  # (-5+10j)
```

## 5. 类型转换

### 什么是类型转换？
将一种数字类型转换为另一种数字类型。

### 转换为整数
```python
x = int(3.9)    # 3（直接截断小数）
y = int("10")   # 10（从字符串转换）

print("int(3.9) =", x)
print("int('10') =", y)
```

### 转换为浮点数
```python
a = float(5)    # 5.0
b = float("3.14")  # 3.14

print("float(5) =", a)
print("float('3.14') =", b)
```

### 转换为复数
```python
z = complex(2)  # (2+0j)
w = complex(3, 4)  # (3+4j)

print("complex(2) =", z)
print("complex(3, 4) =", w)
```

## 6. 算术运算

### 基本运算符
| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| + | 加法 | 5 + 3 | 8 |
| - | 减法 | 10 - 4 | 6 |
| * | 乘法 | 6 * 7 | 42 |
| / | 除法 | 10 / 3 | 3.333... |
| // | 整除 | 10 // 3 | 3 |
| % | 取模 | 10 % 3 | 1 |
| ** | 幂运算 | 2 ** 3 | 8 |

### 运算示例
```python
# 加法
print("5 + 3 =", 5 + 3)

# 减法
print("10 - 4 =", 10 - 4)

# 乘法
print("6 * 7 =", 6 * 7)

# 除法（总是返回浮点数）
print("10 / 3 =", 10 / 3)

# 整除（向下取整）
print("10 // 3 =", 10 // 3)

# 取模（求余数）
print("10 % 3 =", 10 % 3)

# 幂运算
print("2 ** 3 =", 2 ** 3)
```

## 7. 实践练习

```python

area = pi * radius ** 2

print("半径:", radius)
print("面积:", area)
```

### 练习2：温度转换
```python

celsius = 25
fahrenheit = celsius * 9/5 + 32

print(f"{celsius}°C = {fahrenheit}°F")
```

### 练习3：计算平均值

```python
average = (a + b + c) / 3

```

## 8. 常见问题解答

### Q1: 为什么 0.1 + 0.2 不等于 0.3？
**A:** 因为计算机用二进制存储浮点数，有些十进制小数无法精确表示，会产生舍入误差。

### Q2: 整数有大小限制吗？
**A:** 没有！Python的整数可以是任意大小。

### Q3: 如何判断一个数是整数还是浮点数？
**A:** 使用 `type()` 函数：
```python
x = 5
y = 5.0
print(type(x))  # <class 'int'>
print(type(y))  # <class 'float'>
```

### Q4: int(3.9) 为什么是 3 而不是 4？
**A:** `int()` 函数会直接截断小数部分，不是四舍五入。
