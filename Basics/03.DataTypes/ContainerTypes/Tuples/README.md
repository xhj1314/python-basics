# Python 元组教程 

## 1. 什么是元组？

元组是一个有序的、不可变的容器，可以存储多个数据。元组一旦创建，就不能修改。

### 生活中的类比
想象你有一张火车票：
- 车票上的信息（车次、座位号、日期）一旦打印就不能修改
- 元组就像这张车票，创建后内容就固定了

### 创建元组
```python
fruits = ("苹果", "香蕉", "橙子")
print("水果元组:", fruits)
```

## 2. 元组的特点

1. **用小括号 () 创建**
2. **有序**（可以通过索引访问）
3. **不可变**（创建后不能修改）
4. **可以存储不同类型的数据**

### 创建单个元素的元组
```python
# 注意：单个元素必须加逗号
single = (1,)  # 正确
not_tuple = (1)  # 错误！这不是元组，是数字1
```

## 3. 访问元组元素

### 通过索引访问
```python
fruits = ("苹果", "香蕉", "橙子", "葡萄", "西瓜")

print("第一个:", fruits[0])    # 苹果
print("第二个:", fruits[1])    # 香蕉
print("最后一个:", fruits[-1])  # 西瓜
```

### 切片访问
```python
fruits = ("苹果", "香蕉", "橙子", "葡萄", "西瓜")

print("前两个:", fruits[:2])     # ('苹果', '香蕉')
print("后两个:", fruits[-2:])    # ('葡萄', '西瓜')
print("中间三个:", fruits[1:4])  # ('香蕉', '橙子', '葡萄')
```

### 遍历元组
```python
fruits = ("苹果", "香蕉", "橙子")

for fruit in fruits:
    print("水果:", fruit)
```

## 4. 元组操作

### 连接元组
```python
fruits = ("苹果", "香蕉")
more_fruits = ("橙子", "葡萄")
all_fruits = fruits + more_fruits
print(all_fruits)  # ('苹果', '香蕉', '橙子', '葡萄')
```

### 重复元组
```python
repeat = ("哈",) * 3
print(repeat)  # ('哈', '哈', '哈')
```

### 检查元素是否存在
```python
fruits = ("苹果", "香蕉", "橙子")
if "苹果" in fruits:
    print("苹果在元组中")
```

### 获取元素数量
```python
fruits = ("苹果", "香蕉", "橙子")
print("元素数量:", len(fruits))  # 3
```

### 统计元素出现次数
```python
numbers = (1, 2, 3, 2, 2, 4)
print("2出现的次数:", numbers.count(2))  # 3
```

### 查找元素索引
```python
fruits = ("苹果", "香蕉", "橙子")
print("香蕉的索引:", fruits.index("香蕉"))  # 1
```

## 5. 元组解包

### 基本解包
```python
person = ("小明", 18, "北京")
name, age, city = person
print("姓名:", name)  # 小明
print("年龄:", age)   # 18
print("城市:", city)  # 北京
```

### 交换变量
```python
a = 10
b = 20
a, b = b, a  # 使用元组解包交换
print(f"a={a}, b={b}")  # a=20, b=10
```

### 使用*收集多余元素
```python
scores = (90, 85, 92, 78, 88)
first, *middle, last = scores
print("第一个:", first)      # 90
print("中间:", middle)       # [85, 92, 78]
print("最后一个:", last)     # 88
```

## 6. 元组 vs 列表

| 特性 | 元组 | 列表 |
|------|------|------|
| 符号 | () | [] |
| 可变性 | 不可变 | 可变 |
| 速度 | 更快 | 更慢 |
| 用途 | 固定数据 | 可变数据 |

### 什么时候用元组？
- 数据不应该被修改时（如坐标、配置）
- 作为字典的键
- 函数返回多个值

### 什么时候用列表？
- 数据需要修改时
- 需要添加或删除元素时

## 7. 实践练习

### 练习1：坐标点
```python
# 表示一个坐标点（x, y）
point = (3, 4)
x, y = point
print(f"坐标点: ({x}, {y})")

# 计算到原点的距离
import math
distance = math.sqrt(x**2 + y**2)
print(f"到原点的距离: {distance}")
```

### 练习2：一周七天
```python
# 一周的七天（固定不变）
weekdays = ("周一", "周二", "周三", "周四", "周五", "周六", "周日")

# 打印工作日
print("工作日:")
for day in weekdays[:5]:
    print(day)

# 打印周末
print("\n周末:")
for day in weekdays[5:]:
    print(day)
```

### 练习3：函数返回多个值
```python
def get_student_info():
    """返回学生的姓名、年龄、成绩"""
    return "小明", 18, 95.5

name, age, score = get_student_info()
print(f"姓名: {name}, 年龄: {age}, 成绩: {score}")
```

## 8. 常见问题解答

### Q1: 元组和列表有什么区别？
**A:**
- 元组用 `()`，列表用 `[]`
- 元组不可变，列表可变
- 元组更快，列表更慢
- 元组可以作为字典的键，列表不能

### Q2: 为什么要有元组？
**A:**
- 保证数据不被修改
- 作为字典的键
- 函数返回多个值
- 性能更好

### Q3: 元组可以修改吗？
**A:** 不能！元组是不可变的。
```python
fruits = ("苹果", "香蕉", "橙子")
# fruits[0] = "葡萄"  # 错误！不能修改
```

### Q4: 如何"修改"元组？
**A:** 创建一个新的元组：
```python
fruits = ("苹果", "香蕉", "橙子")
new_fruits = fruits + ("葡萄",)  # 创建新元组
print(new_fruits)  # ('苹果', '香蕉', '橙子', '葡萄')
```
