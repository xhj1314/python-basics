# Python 模块教程 

## 1. 什么是模块？

模块是包含Python代码的文件。

## 2. 导入模块

```python
import math
print(math.pi)
```

## 3. 导入特定函数

```python
from math import sqrt
print(sqrt(16))
```

## 4. 导入并重命名

```python
import math as m
print(m.pi)
```

## 5. 常用内置模块

- os - 操作系统接口
- sys - 系统相关
- random - 随机数
- datetime - 日期时间
- json - JSON处理

## 6. 总结

模块让代码更模块化！
